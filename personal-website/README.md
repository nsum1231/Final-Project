# Personal Website

This project is a personal website consisting of multiple pages that showcase an introduction, professional background, and personal interests.

## Project Structure

```
personal-website
├── index.html          # Introductory page with basic information and links to other pages
├── professional.html   # Professional page showcasing resume and work experience
├── hobby.html          # Page dedicated to a personal hobby or passion
├── css
│   └── styles.css      # Styles for the website
├── js
│   └── main.js         # JavaScript functionality for interactivity
├── assets
│   └── resume.pdf      # PDF version of the resume for download
└── README.md           # Documentation for the project
```

## Pages Description

- **index.html**: This is the introductory page of the website. It provides basic information about you and includes links to the professional and hobby pages.

- **professional.html**: This page is dedicated to your professional background. It includes sections for work experience, education, skills, and a link to download your resume.

- **hobby.html**: This page focuses on one of your passions or hobbies. It includes images, descriptions, and relevant information about your interests.

## Setup Instructions

1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the introductory page.
3. Navigate through the links to access the professional and hobby pages.

## Technologies Used

- HTML
- CSS
- JavaScript

Feel free to customize the content and styles to better reflect your personal brand!
